# Nekt: a nova morada do Mago
Plataforma de dados na Nekt

## Workshop gratuito

A partir do dia 17/11 teremos o noss workshop ao vivo criando a nossa plataforma de dados na [Nekt](https://nekt.com/?via=33hoqj8m)!

Nos encontraremos ao vivo na [Twitch](https://twitch.tv/teomewhy) às 9AM para ensinar ao vivo tudo o que você precisa saber sobre a Nekt e criar sua própria plataforma de dados.

## Dados que serão utilizados

No primeiro momento, vamos utilizar os dados abertos que temos no Kaggle:
- [Téo Me Why Loyalty System](https://www.kaggle.com/datasets/teocalvo/teomewhy-loyalty-system)
- [Téo Me Why Education Platform](https://www.kaggle.com/datasets/teocalvo/teomewhy-education-platform)

Ao longo dos próximos meses, pretendemos adicionar novos dados a esta plataforma. Fique a vontade para abrir [Issues](https://github.com/TeoMeWhy/nekt-morada-mago/issues) sugerindo novos datasets.

Ideias atuais:
- [Dota2](https://docs.opendota.com/);
- [Riot](https://developer.riotgames.com/apis);
- [Fogo Cruzado](https://api.fogocruzado.org.br/);
- [Dados Abertos Legislativos do Senado Federal e Congresso Nacional](https://legis.senado.leg.br/dadosabertos/api-docs/swagger-ui/index.html);
- [TSE](https://dadosabertos.tse.jus.br/dataset/);
